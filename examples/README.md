# JARZ-AI Examples

Quick start examples and demo scripts for using the system.

## Available Examples

### API Examples
- Query rental valuations
- Fetch property listings
- Get spatial-temporal predictions

### Demo Workflows
- Single area forecast (e.g., "NW1")
- Compare two districts (e.g., "NW1 vs E14")
- What-if analysis (adjust time horizons and spatial radius)
